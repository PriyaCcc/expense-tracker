# Expense Tracker

A responsive Expense Tracker built using React.js that helps users manage their income and expenses efficiently.
## Features

- Add income and expense transactions
- Delete transactions
- Automatic balance calculation
- Income and expense summaries
- Category-wise transaction tracking
- Search transactions by description
- Local Storage persistence (data remains after refresh)

## Tech Stack

- React.js
- JavaScript
- HTML (JSX)
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/expense-tracker-react.git
```

Go to the project directory:

```bash
cd expense-tracker-react
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open your browser and visit:

```text
http://localhost:3000
```
